#### Introductions
A basic react three project with all needed plugins installed. Recommended for beginners who are attending the Tingeworks ThreeJS course.

- Build Project using\
  ` yarn build`
- Start Development Server\
  ` yarn dev `
- Preview\
  ` yarn preview `
